Privacy Policy for Lalafo DX Assist

Last updated: 2026-03-27

This Privacy Policy explains how the Lalafo DX Assist Chrome extension handles information.

1. What the extension does

Lalafo DX Assist provides QA and developer tools for supported Lalafo domains, including dark mode checks, accessibility scanning, SEO diagnostics, and form-filling helpers.

2. Data collection and use

The extension does not collect, sell, or transfer personal data to external servers.

All extension settings and saved presets are stored locally in your browser using chrome.storage.local.

  • Stored settings include feature toggles and scan preferences.
  • Saved form presets are stored locally only for your own testing workflow.
  • Accessibility and SEO scan results are processed locally in your browser context.

3. Permissions

The extension uses the following Chrome permissions:

  • storage - to save local settings and presets.
  • activeTab - to interact with the currently active tab when you use the extension.
  • scripting - to run local extension scripts on supported pages.

Host permissions are limited to supported Lalafo domains and are used only to enable extension functionality on those domains.

4. Remote code and third parties

The extension does not load or execute remote code for its core functionality. Libraries used by the extension are bundled with the extension package.

The extension may provide clickable links (for example, documentation links), and those links open only when you choose to click them.

5. Data retention and deletion

Data remains in your browser until you remove it.

  • You can clear saved scan results from the extension UI.
  • You can remove all extension data by uninstalling the extension.

6. Security

Since extension data is stored locally in your browser, access to that data is controlled by your browser profile and local device security.

7. Changes to this policy

This policy may be updated from time to time. Any updates will be posted on this page with a revised "Last updated" date.
